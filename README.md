# CrimiReview Landing Page

A clean, minimal landing page for the CrimiReview app - a criminology board exam reviewer.

## Features Showcased

- Adaptive Learning
- Daily Challenges
- Flashcards
- Progress Tracking
- Leaderboard
- Offline Mode

## Tech Stack

- Pure HTML, CSS, JavaScript
- No frameworks or dependencies
- Mobile responsive
- GitHub Pages ready
